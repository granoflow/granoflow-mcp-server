# Learning Task Analysis Profile

Add only the following material to section 8 of the base analysis template:

- capability target: remember, understand, apply, analyze, or create;
- current baseline, difficulty, and prerequisite knowledge;
- available learning material and whether it is sufficient;
- practice, feedback, and mistake-correction loop;
- evidence of independent mastery without answers;
- next review or revalidation trigger.

Learning Evidence must prove changed capability, not merely reading, watching,
note-taking, or feeling familiar. Grill for missing prerequisites, oversized
scope, weak feedback, and self-tests that cannot distinguish recognition from
independent performance.
