# Legacy Learning Plan Profile

This file preserves legacy Plan interpretation. New Work Documents use
`task-analysis-profile-learning.md` as the single conditional learning omission
check across Analysis and applicable Planning.

When reading a legacy learning Plan, preserve these requirements:

- capability progression: baseline, guided practice, independent practice, correction, unprompted test, delayed review;
- AI responsibilities versus learner responsibilities;
- practice and feedback loop;
- independent mastery gate and next review trigger.

Evidence must demonstrate changed capability. Reading, watching, note-taking, or a feeling of understanding is supporting evidence only. Independent practice and manual mastery checks remain user nodes and do not block the AI from preparing later safe materials.
