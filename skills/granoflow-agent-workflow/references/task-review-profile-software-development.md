# Task Review Software Development Profile

When `software_development` applies, attribute diagnosis, implementation,
test/debug, review, rework, release, and acceptance separately. Load the exact
Implementation Learning Ledger used by Delivery. Capture false-success gates,
AI-discovered wrong assumptions, materially abandoned approaches, regression
causes, architecture/process improvements, successful replacement methods, and
reusable engineering knowledge. A clean first pass needs no invented lesson.
Never turn hidden reasoning, routine compiler feedback, or unfiltered logs into
Review content.
